# ブラックジャック


#概要
pythonで作成したブラックジャックゲームの改良版です。10枚のチップを100枚に増やしたらゲームクリア、0枚になってしまったらゲームオーバーです。

#制作目的
・pythonの文法理解
・関数分割による設計力の向上
・読みやすいコードを書く練習

#使用技術
　python 3.x
 randomモジュール

#構成
main():ゲーム全体の制御
  rule():ルール説明
  start():プレイヤーがゲームを始めるための処理
  chip_battle():ゲーム全体の制御
    draw():1枚ドローする処理
    myturn():プレイヤーの処理
    dealersturn():ディーラーの処理
    judgeandchip():勝敗とチップの処理

 #工夫した点
 ・ブラックジャックの配役がでたときの点数倍率の変化を実装
 ・Aの得点を状況に応じて変化させる機能の実装
 ・globalの使用をやめ、引数でデータを渡す設計に変更
 ・関数で役割を分け、可読性が向上

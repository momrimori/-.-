# ブラックジャック


## 概要
Pythonで作成したブラックジャックゲームの改良版です。10枚のチップを100枚に増やしたらゲームクリア、0枚になってしまったらゲームオーバーです。

## 制作目的
- Pythonの文法理解
- 関数分割による設計力の向上
- 読みやすいコードを書く練習

## 使用技術
- Python 3.x
- randomモジュール

## 構成
- `main()`:ゲーム全体の制御
 - `rule()`:ルール説明
 - `start()`:プレイヤーがゲームを始めるための処理
 - `chip_battle()`:0<チップ<=100になるまで行われる処理全体
  - `draw()`:1枚ドローする処理
  - `myturn()`:プレイヤーの処理。2枚引いたのち任意でドロー
  - `dealersturn()`:ディーラーの処理。2枚引いたのち合計17点以上になるまでドロー
  - `judgeandchip()`:勝敗とチップの処理

 ## 工夫した点
 - ブラックジャック成立時の点数倍率(1.5倍)の変化を実装(通常2倍)
 - Aの得点を状況に応じて1点と11点で変化させる機能の実装
 - globalの使用をやめ、引数でデータを渡す設計に変更
 - 関数で役割を分け、可読性を向上
 - ブラックジャック成立判定に変数を導入し、処理の簡潔性と正確性の向上の実現
